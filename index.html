Welcome to the Task Management API

This is a nice simple API myself and my partner built, to manage some simple tasks.
You can create, read, update and delete tasks. Each task has a title, a completed status, a priority level, and a creation timestamp.

API Endpoints


  GET  "/api/tasks"                 //Retrieve all tasks
  GET  "/api/tasks/:id"             //Retrieve a specific task by ID
  POST  "/api/tasks"                //Create a new task
  PUT  "/api/tasks/:id"             //Update an existing task by ID
  DELETE  "/api/tasks/:id"          //Delete a task by ID

Task Object Structure

{
  "id": "string",              // Unique identifier for the task
  "title": "string",           // Title of the task
  "completed": "boolean",      // Completion status of the task
  "priority": "string",        // Priority level of the task (e.g., "high", "medium", "low")
  "created_at": "string"       // Timestamp of when the task was created (ISO 8601 format)
}
